# Botpack-Noob-Black

Noob Black is my Rocket League bot for offline play with RLBot.

# Ancestry:
Noob Black is a direct sequel to Noob Blue, and Noob Yellow before (v4 only). Noob Black has been converted from v4 to v5, making it available in both RLBot versions.

# Skills:
Noob Black's biggest trait is it's teamplay, since it's mainly intended for 3v3 (which is it's best mode), but can also play 2v2 (kinda), mostly useless in 1s.

In the 2025 tournament it was the only python bot in the Platinum bucket.

# Tournaments:
* RLBA
* 2025 RLBot tournament (HC, 2v2), where it was the only python bot in the Platinum bucket and the last python bot eliminated.
* 2025 RLBot tournament afterparty, beating BerroBot (the HC champion) in 3v3.
